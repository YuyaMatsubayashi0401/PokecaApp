============================================================
 ポケカ総合管理システム (Pokeca Manager) セットアップガイド
============================================================

この度は本プロジェクトをインポートしていただきありがとうございます。
本アプリをEclipseで動作させるための手順を以下に記載します。

------------------------------------------------------------
1. プロジェクトのインポート方法
------------------------------------------------------------
① Eclipseを開き、[ファイル] > [インポート] を選択します。
② [一般] > [既存プロジェクトをワークスペースへ] を選択して「次へ」。
③ [アーカイブ・ファイルの選択] にチェックを入れ、本ZIPファイルを選択します。
④ プロジェクト名「PCG_db」が表示されたら「完了」を押してください。

------------------------------------------------------------
2. JDBCドライバ(PostgreSQL)の確認
------------------------------------------------------------
本プロジェクトには接続用のドライバを同梱しています。
もしエラー（赤い感嘆符）が出ている場合は、以下の操作を行ってください。

① プロジェクト内の「lib」フォルダを開く。
② 中にある「postgresql-42.x.x.jar」を右クリック。
③ [ビルド・パス] > [ビルド・パスに追加] を選択。

------------------------------------------------------------
3. データベース(PostgreSQL)の準備
------------------------------------------------------------
本アプリはデータベースを使用します。以下の手順でDBを作成してください。

① PostgreSQL（pgAdmin等）を起動。
② 新規データベース「pokeca_db」を作成。
③ 本アプリのメインメニュー内にある「🛠 セットアップ手順・SQL」ボタン
   からSQLをコピーし、クエリエディタで実行してください。
   ※これにより、decks, deck_cards, storage テーブルが作成されます。

------------------------------------------------------------
4. 接続設定の変更
------------------------------------------------------------
環境に合わせて、接続パスワードを書き換える必要があります。

① 「src/PCG_db/DatabaseManager.java」を開く。
② 13行目付近の「private final String password = "postgrestest";」の
   ダブルクォーテーション内に、ご自身のPostgreSQLのパスワードを入力してください。

------------------------------------------------------------
5. 実行方法
------------------------------------------------------------
① 「src/PCG_db/MainMenu.java」を右クリック。
② [実行] > [Javaアプリケーション] を選択。
③ メインメニューが立ち上がったら「🔌 DB接続テスト」を押し、
   「成功」と表示されれば準備完了です！
